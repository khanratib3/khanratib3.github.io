ESP32 Audio Spectrum Analyzer

Designed ESP32 Audio Spectrum Analyzer with I2S mic and OLED screen to show spectral content up to 20 kHz. Applied FFT on I2S data to convert from time domain with a hamming window function to reduce spectral leakage Determined FFT resolution and bin width to graph on OLED screen using I2C protocol. Verified accuracy of MCU system and graph by comparison with MATLAB FFT simulation

RC Magnetic Field Detecting Robot

Designed and built a remote-controlled robot featuring an RF transmitter for wireless communication, a joystick-controlled drive system using PWM and an H-bridge for speed and direction control, and a custom inductor-based metal detector that varies frequency with coin proximity. Integrated a 2-DOF servo arm to pick up and store detected coins, enabling full autonomous coin retrieval.

Reflow Oven Controller

Programmed a reflow oven controller state machine on an 8051 microcontroller to control a standard toaster oven via a solid-state relay. Utilized timers and PWM for precise temperature control, implemented user programmability for soak and reflow temperature/time settings, added a buzzer to signal process completion, and graphed the entire reflow profile in real time using a Python-based graph.
